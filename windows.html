3.1

Windows 3.1 — La GUI que democratizó el PC

Lanzado en abril de 1992, Windows 3.1 consolidó la interfaz gráfica de Microsoft sobre MS-DOS y vendió más de 3 millones de copias en los primeros dos meses. Introdujo las fuentes TrueType, multimedia y un entorno estable de ventanas superpuestas.

🔍
Escritorio — Administrador de Programas (1992)
🔍
Write, Paintbrush y accesorios clásicos
Año de lanzamiento
Abril 1992
Requisitos mínimos
CPU 286/386SX · 2 MB RAM · 10–15 MB disco · MS-DOS 3.1+
Arquitectura
16 bits con modo mejorado 386 de 32 bits · Dependiente de DOS
Paleta visual
Fondo gris (#C0C0C0) · 16 a 256 colores · Iconos 32×32 px en mapa de bits
⚡ Salto arquitectónico y visual
Primera experiencia gráfica estable sobre DOS para el gran público. El Administrador de Programas con grupos de iconos reemplazó la línea de comandos. Las ventanas usaban un estilo skeuomórfico plano 2D: bordes en relieve simulados con píxeles claros/oscuros, sin aceleración GPU. Se introdujeron las fuentes TrueType (desarrolladas con Apple), eliminando los mapas de bits por tipografías escalables. Soporte de multimedia (MCI), OLE 1.0 para incrustar objetos entre aplicaciones y por primera vez un escritorio visualmente operativo sin tocar el teclado.
TrueType fonts Estilo 2D en relieve Paleta 256 colores OLE 1.0 Multimedia MCI
95

Windows 95 — La revolución del menú Inicio

Lanzado con una campaña histórica con el tema de The Rolling Stones, Windows 95 vendió más de 7 millones de copias en 5 semanas. Introdujo el Menú Inicio, la barra de tareas, Plug and Play y la API Win32 completa, cambiando para siempre cómo el mundo interactúa con una computadora.

Escritorio predeterminado con barra de tareas (1995)
🔍
El icónico Menú Inicio — debut en 1995
Año de lanzamiento
Agosto 1995
Requisitos mínimos
386 DX · 4 MB RAM · 55–65 MB disco · CD-ROM recomendado
Arquitectura
Híbrido 16/32 bits · Kernel mejorado con compatibilidad DOS · Win32 API
Paleta visual
Fondo teal (#008080) · Gris (#C0C0C0) en ventanas · Iconos 32×32 con profundidad 3D simulada
⚡ Salto arquitectónico y visual
El salto visual más grande hasta entonces: el Menú Inicio y la barra de tareas reemplazaron al Administrador de Programas. Las ventanas adoptaron un estilo "3D" más pronunciado con botones Minimizar/Maximizar/Cerrar en la esquina superior derecha (el trío que persiste hasta hoy). Se introdujo el Explorador de Windows con vista de árbol de directorios. Los iconos ganaron mayor profundidad visual con degradados sutiles. Plug and Play permitió detectar hardware automáticamente. La API Win32 unificó el modelo de programación, permitiendo aplicaciones de 32 bits plenas como Microsoft Office 95.
Menú Inicio Barra de tareas Win32 API Plug and Play Explorer shell Botones Min/Max/Cerrar
98

Windows 98 — Windows se conecta a Internet

Windows 98 refinó el escritorio para la era web, integró Internet Explorer en el shell del sistema y fue el primer Windows en soportar múltiples monitores, USB nativo y reproducción de DVD de forma oficial.

🔍
Escritorio con barra de inicio rápido (1998)

Junio 1998 · SE: Mayo 1999
Requisitos mínimos
486 DX2/66 · 16 MB RAM · 195 MB disco
Arquitectura
Híbrido 16/32 bits basado en DOS · Kernel 98 mejorado · FAT32 nativo
Paleta visual
Idéntica a Win95 visualmente · Active Desktop añadía contenido web como fondo
⚡ Salto arquitectónico y visual
Aunque visualmente muy similar a Windows 95, el salto real fue la integración profunda de Internet Explorer en el shell: el Explorador de Windows podía navegar páginas web y el Active Desktop permitía incrustar contenido HTML directamente en el fondo del escritorio. Se añadió la barra de inicio rápido junto al botón Inicio. Primer Windows con soporte nativo de USB, múltiples monitores y DVD. Las carpetas podían mostrar vistas personalizadas con HTML. La paleta de 24 bits (TrueColor) se volvió estándar, permitiendo iconos y fondos de mayor calidad visual.

ME

Windows Millennium Edition — El último heredero del DOS

Lanzado en septiembre del año 2000, el mismo año que Windows 2000, Windows ME fue el último sistema de la línea Windows 9x — la rama doméstica basada en DOS. Aunque compartían año de lanzamiento, ME y Windows 2000 eran productos completamente distintos, con kernels diferentes, públicos distintos y destinos opuestos.

⚠️ ¿ME y Windows 2000 eran lo mismo?
No. Convivieron en el mismo año pero eran sistemas completamente distintos:
• Windows ME → kernel híbrido DOS/9x · para usuarios domésticos · heredero de Win98
• Windows 2000 → kernel NT puro · para empresas y profesionales · heredero de NT4
La confusión era común porque Microsoft los lanzó con pocos meses de diferencia. Windows XP (2001) fue el primero en unificar ambas ramas bajo un único kernel NT para todos los públicos.

Escritorio de Windows ME (2000)
🔍
Movie Maker y Media Player 7 — novedades clave
Año de lanzamiento
Septiembre 2000 · Mismo año que Windows 2000 (febrero)
Requisitos mínimos
Pentium 150 MHz · 32 MB RAM · 480–645 MB disco
Arquitectura
Híbrido 16/32 bits · Línea Windows 9x · Eliminó el modo real de DOS · NO era kernel NT
Paleta visual
Idéntica a Win98 visualmente · Iconos de mayor calidad · Interfaz heredada del diseño 9x · Sin cambios de lenguaje de diseño
⚡ Salto arquitectónico y visual
ME fue el intento de modernizar la rama 9x antes de que XP la reemplazara definitivamente. Visualmente era casi idéntico a Windows 98: mismo gris, misma barra de tareas, mismos iconos — aunque con mayor calidad en los fondos de pantalla predeterminados. El verdadero cambio fue funcional: eliminó el modo real de DOS (aunque la base interna del kernel seguía siendo DOS), introdujo Windows Movie Maker (el primer editor de video básico integrado en Windows), Windows Media Player 7 con nueva interfaz, soporte de USB mejorado y restauración del sistema nativa (System Restore) por primera vez. Sin embargo, fue el Windows más criticado de la historia por su inestabilidad: muchos PCs que corrían bien Win98 fallaban con ME por incompatibilidades de drivers. Fue discontinuado en 2006 y considerado el último eslabón de la era DOS — una era que terminó definitivamente con la llegada de Windows XP en 2001.

2000

Windows 2000 — El kernel NT llega al escritorio profesional

Windows 2000 trasladó la robustez del kernel NT4 al escritorio profesional, con soporte completo para NTFS, Active Directory, seguridad avanzada y gestión de redes. No era para uso doméstico — ese rol lo cubría Windows ME ese mismo año. Fue el puente directo hacia Windows XP.

Febrero 2000
Requisitos mínimos

Arquitectura
NT 5.0 · 32 bits (versiones 64-bit en edición Server) · Kernel NT puro
Paleta visual
Heredada de Win98 · Iconos refinados · Mayor calidad en fondos · Fondo predeterminado azul oscuro
⚡ Salto arquitectónico y visual
El fin definitivo del legado MS-DOS en el escritorio. Windows 2000 fue el primer Windows basado 100% en el kernel NT para uso general profesional, con NTFS, seguridad por usuarios y grupos, y soporte de Active Directory. Visualmente adoptó una interfaz más pulida que Win98/95: iconos de mayor resolución, ventanas ligeramente más refinadas y un escritorio más profesional. Introdujo el MMC (Microsoft Management Console) para administración centralizada. La estabilidad mejoró radicalmente — el temido "pantallón azul" se volvió mucho menos frecuente. Sentó las bases directas para Windows XP.
Kernel NT puro NTFS Active Directory Sin DOS MMC Seguridad por usuarios
XP

Windows XP — El mayor salto visual de su era

Con más de 400 millones de copias vendidas y soporte extendido hasta 2014, Windows XP fue el sistema operativo más popular de la historia por años. Unió la base NT estable con una interfaz radicalmente renovada y colorida que definió una generación entera.

🔍
Tema Luna con el icónico fondo "Bliss" (2001)
🔍
Nuevo Menú Inicio con dos columnas (SP3)
Año de lanzamiento
Octubre 2001 · SP3: Abril 2008
Requisitos mínimos
233 MHz · 64 MB RAM · 1.5 GB disco · Tarjeta de video SVGA
Arquitectura
NT 5.1 · 32 bits y 64 bits (x64 Edition) · Kernel NT unificado
Paleta visual
Azul luna (#1C5FA8) · Verde brillante · Botón Inicio verde · Iconos 48×48 px con sombras · ClearType
⚡ Salto arquitectónico y visual
El mayor rediseño visual hasta ese momento. El tema Luna reemplazó el gris de Windows 95/98/2000 con colores vivos: barra de tareas en azul brillante, botón Inicio verde con logo y texto, bordes de ventana redondeados y un fondo de pantalla predeterminado ("Bliss" — la fotografía más reproducida de la historia). Los iconos saltaron a 48×48 px con suavizado y sombras suaves. Se introdujo ClearType para mejor renderizado de texto en LCD. El Menú Inicio fue rediseñado con dos columnas. Se unificó el sistema de consumo y empresarial sobre un único kernel NT, eliminando la línea Windows 9x para siempre. Tres temas de color: Luna Azul, Olivo y Plata.
Tema Luna Fondo Bliss ClearType Iconos 48×48 NT unificado DirectX 8/9
Vista

Windows Vista — Aero Glass y el futuro visual

Tras 5 años de desarrollo (el más largo entre versiones de Windows), Vista introdujo la interfaz Aero Glass con transparencias reales, el UAC de seguridad, la nueva pila de controladores y DirectX 10. Revolucionario en diseño pero criticado por su rendimiento inicial.

🔍
Aero Glass con barra lateral de gadgets (2007)
🔍
Flip 3D — conmutador de ventanas en cascada
Año de lanzamiento
Enero 2007 (empresas: Noviembre 2006)
Requisitos mínimos
800 MHz · 512 MB RAM · 15 GB disco · GPU con DirectX 9 + 128 MB VRAM para Aero
Arquitectura
NT 6.0 · 32 y 64 bits · WDDM 1.0 · Nuevo modelo de drivers
Paleta visual
Aero Glass: transparencias con blur real · Iconos vectoriales hasta 256×256 px · Tipografía Segoe UI
⚡ Salto arquitectónico y visual
El mayor salto visual desde XP. Aero Glass introdujo transparencias reales en los bordes de ventana con efecto blur del contenido detrás — la primera vez que Windows usó la GPU para renderizar la interfaz del sistema operativo completa (vía el nuevo WDDM — Windows Display Driver Model). Los iconos escalaban hasta 256×256 px con suavizado total. Se introdujo la tipografía Segoe UI que perduraría hasta Windows 11. El Flip 3D mostraba ventanas en cascada tridimensional. La barra de tareas mostraba miniaturas en vivo al pasar el mouse. El UAC (Control de Cuentas de Usuario) y la firma digital de drivers reforzaron la seguridad. DirectX 10 fue exclusivo de Vista.
Aero Glass WDDM Segoe UI Iconos 256×256 Flip 3D UAC DirectX 10
7

Windows 7 — Aero perfeccionado

Windows 7 pulió cada aspecto de Vista, redujo el consumo de recursos, refinó la interfaz Aero y se convirtió en la versión más aclamada de Windows en la era NT moderna, manteniendo su popularidad hasta bien entrada la era de Windows 10.

🔍
Escritorio con nueva barra de tareas (2009)
🔍
Aero Snap — división de pantalla inteligente
Año de lanzamiento
Octubre 2009
Requisitos mínimos
1 GHz · 1 GB RAM (32-bit) / 2 GB (64-bit) · 16/20 GB disco
Arquitectura
NT 6.1 · 32 y 64 bits · WDDM 1.1 · DirectX 11
Paleta visual
Aero refinado · Barra de tareas translúcida elegante · Sin sidebar · Themes con colores de acento
⚡ Salto arquitectónico y visual
Refinamiento total de Vista sin romper nada. La barra de tareas fue completamente rediseñada: iconos más grandes sin etiqueta de texto, miniaturas en vivo agrupadas y soporte de Jump Lists (menú contextual con acciones frecuentes). Se eliminó la barra lateral, los gadgets pasaron al escritorio libremente. Aero Peek transparentaba todas las ventanas para ver el escritorio. Aero Snap organizaba ventanas arrastrando al borde. Aero Shake minimizaba todas las ventanas agitando una. Soporte mejorado de pantallas táctiles multi-touch. DirectX 11 llegó con tesselation y shaders de cómputo. Fue el Windows más querido por su equilibrio visual-rendimiento.
Aero Peek Aero Snap Jump Lists Multi-touch DirectX 11 Taskbar rediseñada
8

Windows 8 — El gran experimento táctil

Windows 8 fue la apuesta más radical de Microsoft: eliminar el menú Inicio clásico y reemplazarlo con una pantalla de tiles para dispositivos táctiles. Introducía soporte ARM, UEFI nativo y un nuevo paradigma de apps modernas — pero dividió a los usuarios.

🔍
Pantalla Metro/Modern UI con tiles (2013)
🔍
Escritorio clásico con Charms Bar lateral
Año de lanzamiento
Octubre 2012 · Windows 8.1: Octubre 2013
Requisitos mínimos
1 GHz · 1 GB RAM (32-bit) / 2 GB (64-bit) · 16/20 GB disco
Arquitectura
NT 6.2/6.3 · 32 y 64 bits · Windows RT para ARM · UEFI/Secure Boot · WinRT API
Paleta visual
Diseño Flat total: colores sólidos y saturados · Sin transparencias · Tipografía Segoe WP · Sin Aero
⚡ Salto arquitectónico y visual
La ruptura más radical desde Windows 95. Se eliminaron las transparencias Aero y se adoptó el lenguaje de diseño Metro (renombrado "Modern UI"): colores planos y sólidos, sin sombras, sin gradientes — el primer Windows con diseño completamente flat. La pantalla de inicio con tiles animados reemplazó al menú Inicio. La Charms Bar aparecía deslizando desde la derecha. Las apps modernas corrían en pantalla completa. Se introdujo Windows RT para procesadores ARM. La API WinRT permitía apps tanto en tablets como en PCs. El escritorio clásico coexistía con la interfaz táctil, creando confusión. Windows 8.1 reintrodujo el botón Inicio (aunque llevaba a la pantalla táctil).
Metro / Modern UI Diseño Flat ARM (Windows RT) UEFI/Secure Boot WinRT API Charms Bar
10

Windows 10 — Windows como servicio

Windows 10 reconcilió el escritorio clásico con las ideas modernas de Windows 8 y adoptó el modelo de Windows como Servicio con actualizaciones continuas. Se volvió el sistema operativo más usado de la historia, declarado inicialmente "el último Windows".

Escritorio con menú Inicio híbrido (2015)

Centro de actividades y notificaciones
Año de lanzamiento
Julio 2015 · Fin de soporte: Octubre 2025
Requisitos mínimos
1 GHz · 1 GB RAM (32-bit) / 2 GB (64-bit) · 16/20 GB disco
Arquitectura
NT 10.0 · 32 y 64 bits · OneCore unificado · WSL (Linux subsystem)
Paleta visual

⚡ Salto arquitectónico y visual
Vuelta al menú Inicio pero evolucionado: lista de apps + tiles opcionales. Se introdujo el Fluent Design System de forma progresiva: material Acrílico (translúcidez difuminada), efectos de revelado (iluminación al pasar el ratón), profundidad y sombras suaves. Modo oscuro nativo para el sistema y apps. El Centro de Actividades lateral reemplazó los Charms. DirectX 12 optimizó gráficos multi-núcleo. El núcleo OneCore unificó Windows en PC, Xbox, IoT y HoloLens. El WSL (Subsistema de Windows para Linux) permitió correr Linux nativo. Cortana integrada en la barra de búsqueda. Windows Hello introdujo autenticación biométrica. Trece actualizaciones principales en 6 años.
Fluent Design Material Acrílico Modo oscuro DirectX 12 OneCore WSL Windows Hello
11

Windows 11 — El rediseño más ambicioso desde Vista

Lanzado en octubre de 2021 y superando el 51% del mercado de escritorio en 2025, Windows 11 es el rediseño visual más profundo desde la era Aero. Centra el menú Inicio, redondea todo el sistema, refuerza la seguridad con TPM 2.0 e integra IA con Copilot.

🔍
Escritorio con menú Inicio centrado (2021)
🔍
Menú Inicio centrado sin tiles (2021)
Año de lanzamiento
Octubre 2021 · Cuota de mercado: 51.8% (julio 2025)
Requisitos mínimos
1 GHz dual-core 64-bit · 4 GB RAM · 64 GB disco · TPM 2.0 · Secure Boot · DirectX 12
Arquitectura
NT 10.0 (build 22000+) · Solo 64 bits · TPM 2.0 obligatorio · DirectStorage · WSL 2 mejorado
Paleta visual
Mica + Acrílico · Bordes redondeados globales · Iconos rediseñados planos-profundos · Segoe UI Variable · Menú centrado
⚡ Salto arquitectónico y visual
El rediseño más coherente y ambicioso desde Windows Vista. Bordes redondeados en absolutamente todo: ventanas, menús, botones, notificaciones. Dos nuevos materiales de UI: Mica (translúcido con tinte del fondo del escritorio) y Acrílico (blur del contenido detrás). Los iconos fueron completamente rediseñados con una nueva estética plana-expresiva y paleta de colores actualizada. La barra de tareas se centró (polémica pero coherente con la tendencia moderna). La tipografía Segoe UI Variable adapta su grosor automáticamente al tamaño. El menú Inicio eliminó los tiles (diseño limpio). DirectStorage redujo tiempos de carga en juegos. Auto HDR mejoró el color en pantallas compatibles. TPM 2.0 obligatorio elevó el estándar de seguridad. Integración de Copilot (IA) desde 2023. Solo 64 bits — adiós definitivo al legado de 32 bits.
Material Mica Bordes redondeados Segoe UI Variable TPM 2.0 DirectStorage Auto HDR Copilot IA Solo 64-bit
